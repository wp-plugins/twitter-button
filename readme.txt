== Description ==

**Twitter Button** is a WordPress plugin that allows you to easily integrate **[Tweet Button](http://twitter.com/goodies/tweetbutton)** in your blog. 


**Features**

* Easy customization
* 3 methods to include Twitter Button in your blog - automatic, manual or by shortcode
* Exclude Twitter Button from certain pages (admin option)
* Language support
* Restore option

== Installation ==

1. Unpack the download-package.
2. Upload '**twitter-button**' folder to the '**/wp-content/plugins/**' directory
3. Activate the plugin through the '**Plugins**' menu in WordPress
4. Done and Ready.

[More information](http://www.svil4ok.com/wordpress-plugins/twitter-button/#installation)
