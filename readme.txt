== Description ==

**Twitter Button** is a WordPress plugin that allows you to easily integrate **[Tweet Button](http://twitter.com/goodies/tweetbutton)** in your blog. It provides a live count of how many times your post/page has been retweeted throughout Twitter.


**Features**

* Easy customization
* 3 method to include Twitter Button in your blog - automatic, manual or by shortcode
* Language support
* Restore option


**Translations**

== Installation ==

1. Unpack the download-package.
2. Upload 'twitter-button' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Done and Ready.

[More information](http://www.svil4ok.com/wordpress-plugins/twitter-button/#installation)
